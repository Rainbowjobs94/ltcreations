## First response rule
1. Assign task ID.
2. Confirm request in one sentence.
3. Ask up to two questions:
   - What should final output look like?
   - Where should output be delivered/stored?
4. Propose a default final output so work starts immediately.
5. Begin first deliverable immediately.

## Standard deliverables
1. **Chat Map**
   - Chat title/date
   - Goal
   - Decisions
   - Pages/features/copy/protocols
   - Current status
   - Missing pieces
   - Next actions

2. **Regression Report**
   - Intended vs currently missing
   - Severity
   - Recommended fix

3. **Update Site Patch Plan**
   - Target site + page
   - Section delta
   - New copy block
   - UX delta
   - Implementation steps
   - QA checklist

4. **Recovery Pack**
   - Recovered copy
   - Recovered page outline
   - Recovered feature spec
   - Recovered CTA/link map
   - Rollback notes
